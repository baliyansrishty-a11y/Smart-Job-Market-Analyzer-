# Smart Job Market Analyzer

## Overview
A data analysis project that analyzes job market trends using Python and machine learning.

## Features
- Job data cleaning
- Skill demand analysis
- Salary trend analysis
- Visualization
- Market insights

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## How to Run

Install dependencies:

pip install -r requirements.txt

Run notebook:

jupyter notebook
